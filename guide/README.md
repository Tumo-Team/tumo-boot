## Deploy

本文档基于 [docsify](https://docsify.js.org/#/) ，推荐以下两种部署方式：

1. 使用 Github Pages 部署文档（缺点：国内访问速度慢）
2. 使用自己的服务器，部署到Nginx（缺点：需要手动执行脚本）

详细请看文档：[文档部署](docs/other/deploy.md)
